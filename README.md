## 📌 Overview

This project demonstrates a complete **Data Analytics workflow**, starting from raw customer data and ending with an interactive Power BI dashboard.

The project covers:

- 📥 Loading the dataset
- 🔍 Exploratory Data Analysis (EDA)
- 🧹 Data Cleaning & Preprocessing
- 🐘 SQL Analysis using PostgreSQL
- 📊 Interactive Power BI Dashboard
- 📄 Business Report
- 🎯 Presentation using Gamma

The goal is to transform raw customer data into meaningful insights that help understand customer behavior and support data-driven business decisions.

## 🚀 How to Run

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/princegupta0030/Customers-Data-Analysis-and-Dashboard-Project.git
```

### 2️⃣ Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

### 3️⃣ Run the Python Notebooks

```bash
jupyter notebook
```

Execute:

- `data_cleaning.ipynb`
- `eda.ipynb`
- `analysis.ipynb`

### 4️⃣ Import Data into PostgreSQL

- Create a PostgreSQL database.
- Import the cleaned dataset.
- Execute the SQL queries available in the `SQL` folder.

### 5️⃣ Open the Power BI Dashboard

Open:

```text
Dashboard.pbix
```

Interact with filters, slicers, and visualizations.
